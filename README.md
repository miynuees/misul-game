# 조형 요소·원리 월드컵 게임

미술 수업용 실시간 도구. 17개 조형 키워드 × 49개 소재로 도형 표현 후 이상형 월드컵 토너먼트로 우승작을 뽑아요.

## 사용 방법

- **선생님**: `teacher.html` 열고 방 만들기 → 코드 학생들에게 공유 → 키워드/소재 뽑기 → 시작
- **학생**: `index.html` 열고 이름·방 코드 입력 → 3분 동안 도형으로 표현 → 제출
- **선생님**: 자동 진행되는 토너먼트에서 매 매치 승자 선택 → 우승!

## 파일

- `index.html` — 학생용 편집기
- `teacher.html` — 선생님용 콘솔
- `firebase-config.js` — Firebase 설정 (공개 가능, 보안 규칙으로 보호됨)
